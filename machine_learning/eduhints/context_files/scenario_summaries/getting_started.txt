The tasks involve pwd, ls, man, cd, mv, cp, chmod, md5sum, grep, find and others. Key objectives include understanding command syntax, file manipulation, and permission settings, using man for command documentation, and composing commands with redirection and filters such as less and sort. They explore options for ls to display hidden files. The student must rename and copy files using mv and cp, adjust file permissions with chmod, and use pattern matching with both regular expressions and glob patterns.
