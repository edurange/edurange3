FILE_WRANGLER: The student navigates through tasks involving the pwd, ls, man, cd, mv, cp, and chmod commands. Key objectives include understanding command syntax, file manipulation, and permission settings. The student learns to use pwd for directory location, ls for listing files, and man for command documentation. They explore options for ls to display hidden files and learn to change directories with cd. The student practices renaming and copying files using mv and cp, respectively, and adjusts file permissions with chmod. Throughout the module, the student answers questions to reinforce their understanding of each task, culminating in a comprehensive exercise that combines multiple commands and concepts.
