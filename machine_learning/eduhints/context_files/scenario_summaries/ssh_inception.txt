SSH_INCEPTION: The student is tasked with completing a series of challenges involving SSH (Secure Shell) and other security-related concepts. These tasks include using SSH via command line to access different levels, understanding encryption, utilizing man pages and SSH, performing subnet scans and logging into IPs, public/private key encryption, searching the local filesystem, using FTP, decoding encrypted files, and finally, escaping a sinister-looking host known as Satan's Palace. Throughout these tasks, the student must learn and apply various techniques and commands to progress, with the ultimate goal of becoming the ultimate SSH master.
